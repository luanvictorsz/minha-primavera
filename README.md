# Projeto Pessoal – Demonstração de Afeto

Este é um projeto pessoal desenvolvido com o objetivo de demonstrar carinho e proximidade com minha namorada durante a semana, especialmente nos dias em que não conseguimos nos ver pessoalmente.

A proposta do projeto é reunir pequenos elementos da nossa convivência em uma experiência interativa e significativa, funcionando como um espaço digital com lembranças, mensagens e momentos que representam a nossa relação.

---

## Ideia do Projeto

Criar um ambiente interativo que represente a nossa história, nossos hábitos e nossa conexão, utilizando recursos simples, porém simbólicos.

É basicamente um “cantinho virtual” que transmite afeto através de:
- lembranças
- mensagens
- brincadeiras
- momentos marcantes

---

## Funcionalidades Atuais

Atualmente, o projeto conta com as seguintes funcionalidades:

### Jogos
Pequenos jogos simples para entretenimento e momentos de diversão compartilhada.

### Sistema de Frases Aleatórias
Um sistema que exibe frases reais ditas por nós durante nossos encontros, trazendo lembranças espontâneas e significativas.

### Simulador de WhatsApp
Uma recriação da nossa **primeira conversa**, simulando a interface de um chat, para relembrar o início de tudo.

### Página "Sobre"
Uma página contendo informações sobre o projeto, a ideia por trás dele e o propósito emocional — semelhante ao conteúdo deste README.

---

## Objetivo

Mais do que um projeto técnico, este projeto é um gesto de carinho.

A ideia é transformar pequenos momentos do nosso dia a dia em algo permanente, acessível e especial — principalmente nos dias em que a distância existe.

---

## Futuras Ideias

Algumas possíveis melhorias para o projeto:

- Linha do tempo com momentos importantes
- Galeria de fotos e memórias
- Sistema de cartas/mensagens programadas
- Pequenos desafios ou interações personalizadas
- Música ou trilha sonora marcante para nós

## Status do Projeto

Este projeto não tem fins comerciais.
